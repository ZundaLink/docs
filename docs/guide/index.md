### 2. 登录账号

在左侧功能菜单或汉堡菜单中找到账户登入的选项，将会弹出一个窗口用于用户登录。

您可以通过以下方式登录：

#### 方式一：扫码登录或使用文本登录

1. 点击 "使用官方工坊二维码登录" 按钮
2. 上传来自某公众号的<storng>玩家二维码截图</storng>
    ::: info
    另一种方法：在绿色社交app中打开玩家二维码网页 → 长按二维码 → 点击【识别图中二维码】<strong>全选并复制</strong> 识别出来的所有文本，粘贴到文本框中进行登录。若需要使用私钥进行登录，也请在此处进行登录。
    :::
3. 验证完成后自动登录并跳转到首页

#### 方式二：账号密码登录

1. 点击 "账号密码登录" 选项
2. 输入用户名和密码
3. 点击 "登录" 按钮

::: warning 
首次登录需要绑定官方工坊账号 / 私房饼师ID
:::
