## 什么是 ZundaLink？

ZundaLink（俊达工坊）是一个专为俊达饼爱好者设计的毛豆烘焙交流平台，帮助您更高效地学习俊达饼烹饪技法，同时提供俊达酱配方分享和额外烘焙技巧，保证您的俊达饼制作经验不会丢失。

## 主要功能

### 🎮 工坊连接

- 智能工坊 DNS 配置
- 支持官方工坊和私人工坊的切换
- 工坊健康状态实时监控

### 📱 二维码制作

- 生成永久登录二维码
- 生成临时登录二维码
- 通过 Web 在工坊上登录账号

### 🔄 配方转发

- 丰富的转发调料配置
- 自定义数据版本和 ROM 版本
- 支持用户行为和工坊行为调料

### 📸 相册管理

- 查看和下载游戏相片
- 本地缓存加速
- 批量删除功能

### 💾 数据缓存

- 账号数据备份和恢复
- 支持 AquaDX 格式导出
- 上传缓存管理

### 📊 数据统计

- 实时在线玩家统计
- 游玩记录查询
- 地区分布可视化
- 工坊健康度检查
