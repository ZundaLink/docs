# 使用已 Root 的 Android 手机安装（高级）

本文档介绍如何在已 Root 的 Android 设备上配置 ZundaLink 服务。

Android 手机安装方式适合以下场景：

- 需要移动便携的代理方案

此安装方式需要设备已获取 Root 权限。
