# 加入 ZundaLink

本文档介绍如何申请 ZundaLink 使用权限。

在开始之前，请确认您已经获得机台所有者的同意，并向管理员提交包括但不限 ZundaLink 的使用条款、用户知情同意书以及机台的部分唯一资料等资料。

::: tip 注意
ZundaLink 的加入目前为邀请制，如果您有计划加入 ZundaLink ，请提供邀请者的信息。
:::



### 准备申请材料

请各位用户按要求准备并提交以下证明材料（包括但不限于）：

1. 机厅资质相关证明：需明确所使用机厅非某大型连锁品牌机厅，且该机厅对本类服务拥有知情权、知晓机台可使用特殊服务，并要求店长同意本服务的[**用户协议**和**免责声明**](../install/user-rules.md)。
2. 身份验证相关照片：**机台Keychip号和ACA号直摄照片、机厅直摄照片、机台直摄照片（以上照片用于身份核验，需保证清晰可辨）**。
3. 使用许可相关证明：
    - 店长（非机修人员）准许使用本服务的证明图；
    - 机厅玩家内部允许继续使用本服务的投票证明图。
4. 其他补充证明材料：与上述事宜相关的其他有效证明材料。
5. 特殊情况说明：若所使用机台为个人所有，需额外提供机台归属权的证明材料。

### 提交材料审核

将准备好的材料发送至 [@McDLdshEartBot](https://t.me/McDLdshEartBot) ，并附加邀请者的信息以供审核。

提交审核材料后，管理员会一般会在 1-3 个工作日内处理您的申请。

### 获取访问权限

审核通过后，您将收到：

- 工坊私钥
- 服务器地址
- DNS 配置信息

收到上述关键信息后，您可以查阅[快速开始指南](../guide/)以更快地上手 ZundaLink 服务。

::: warning 如果遭到拒绝
一般来讲会写明拒绝原因。若是需要补充某些遗漏的信息或附加证明材料，请在申请后的12个自然日内完成证明材料提交。
:::

## 常见问题

### Q: 审核需要多长时间？

A: 通常为 1-3 个工作日。如遇节假日或申请量较大，可能会有延迟。

### Q: 可以申请多个账号吗？

A: 原则上一个工坊一个账号。
