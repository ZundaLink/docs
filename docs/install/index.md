# 安装概述

欢迎使用 ZundaLink！本章节将指导您完成系统的安装和配置。

## 提交表单

加入 ZundaLink 需要先提交审核表单，等待管理员审核通过后获得访问权限。

[提交审核表单](./submit-form)

## 安装方式

ZundaLink 提供多种安装方式，您可以根据自己的设备和网络环境选择最适合的方式：

### 1. [连接路由器安装](./router)

- 适用于：拥有 OpenWRT/Padavan 等智能路由器的用户
- 难度：⭐⭐
- 描述：通过路由器 DNS 劫持工坊 DNS 服务器实现安装

### 2. [使用U盘安装](./usb)

- 适用于：Windows 系统用户
- 难度：⭐⭐
- 描述：通过插入 U 盘到官方工坊安装 ZundaLink

### 3. [使用已 Root 的 Android 手机安装（高级）](./android-root)

- 适用于：拥有已 Root 的 Android 设备的用户
- 难度：⭐⭐⭐
- 描述：在 Android 设备上运行 DNS 劫持工具实现安装

## 系统要求

- **必须为官方工坊，不能使用第三方工坊安装**
- 网络连接：稳定的互联网连接

## 准备工作

在安装之前，请确保您已完成以下准备工作：

1. **注册账号**
   - 访问 ZundaLink Web 界面
   - 通过二维码或用户名密码注册账号

2. **了解网络环境**
   - 确认您的网络类型，检查是否有防火墙或网络限制
   - 测试与 ZundaLink 服务器的连通性

3. **确认来自工坊主人的许可**
   - 确认您连接到 ZundaLink 的行为是否已获得工坊主人的授权许可，并获得其书面\电子授权许可说明。

## 常见问题

### Q: 哪种安装方式最好？

A: 哪种方便就选择哪种方式安装。

### Q: 安装后无法连接怎么办？

A: 首先确认工坊是否处于VPN模式下，若处于则测试使用公网端点；若公网端点仍然出现问题，请联系 ZundaLink 管理员以获取更多帮助。
